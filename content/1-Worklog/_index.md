**Week 1:** [Getting familiar with AWS and basic AWS services](1.1-week1/)

**Week 4:** [Deep dive into IAM + introduction to Lambda and API Gateway](1.4-week4/)

**Week 5:** [Learning Lambda and DynamoDB basics with read/write operations](1.5-week5/)

**Week 6:** [Building HTTP API integrations (API Gateway → Lambda → DynamoDB)](1.6-week6/)
