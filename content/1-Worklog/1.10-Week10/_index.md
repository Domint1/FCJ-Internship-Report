### Week 10 Objectives:

* Learn the basics of AWS CloudWatch for monitoring cloud applications.
* Understand CloudWatch Logs and how Lambda/API Gateway send log output.
* Learn CloudWatch Metrics (CPU, network, error count, latency).
* Create simple CloudWatch Alarms for essential monitoring.
* Understand how monitoring helps detect issues in serverless applications.
