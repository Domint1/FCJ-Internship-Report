### Week 7 Objectives:

* Learn S3 in more depth (buckets, objects, permissions).
* Understand how S3 static hosting works.
* Learn CloudFront fundamentals for delivering static content globally.
* Practice hosting a simple static site on S3.
* Connect S3 → CloudFront to improve performance and security.
